# Содержимое папки src/ 

Здесь описано все содержимое папки *src/* с пояснениями, за что отвечает каждый компонент папке, а также за что отвечает каждый файл внутри компонента

## Components

### Описание

Папка содержит внутри себя компоненты (чаще всего секции), которые в последствии подключаются к основным страницам.

Подключение к странице осуществляется следующим образом.

```html
<!--=include название.компонента.html -->
```

Имя папки в которой содержится компонент, должно совпадать с названием самого компонента.

Каждая папка компонента содержит внутри html компонент (разметку) и в большинстве случаях scss-файл (стиль)
Логика в компонентах отсутствует.

### Дерево папки components
```yml
- components/
    Главные задние фоны (background.hero) >>>
    - background.hero.cases/ - для страницы кейсов
    - background.hero.main/ - для основной страницы
    - background.hero.services/ - для страницы сервисов

    - footer/ - "подвал" сайта (футер)
    - header/ - шапка сайта

    Модальные окна >>>
    - modal.form/ - окно формы

    Основные секции >>>
    - section.about-us/ - секция "О нас"
    - section.big-form/ - секция с большой формой
    - section.cases/ - секция "Кейсы"
    - section.clients/ - секция "Клиенты"

    ----Первые (главные) секции (section.hero)----
    - section.hero.cases/ - для страницы кейсов
    - section.hero.main/ - для главной страницы
    - section.hero.services/ - для страницы сервисов

    - section.main-video/ - секция с основным видео
    - section.partners/ - секция "Партнеры"
    - section.questions/ - секция с вопросами
    - section.reviews/ - секция "Отзывы"
    - section.servicesLarge/ - секция сервисов (крупные карточки)
    - section.servicesSmall/ - секция сервисов (малые карточки)
    - section.slider/ - секция со слайдером
    - section.small-form/ - секция с малой формой

    Devs-компоненты >>>
    - uikit.buttons/ - компонент с кнопками
```



## Fonts 

### Описание

Папка содержит внутри себя основные шрифты, используемые в проекте.

Используемые шрифты:
* Inter
* Lato (Regular)
* Manrope
* Poppins (Medium)

## Images

### Описание

Папка содержит изображения, используемые в проекте.

Основной формат изображений - *.svg*, *.webp*

### Дерево папки images
```yml
- images/
    - background/ - задние фоны (сейчас только секции hero)
    - cards/ - изображения для карточек
    - icons/ - мелкие иконки
```

## JS 

### Описание

Папка содержит файлы javascript формирующее логическую работу проекта.

### Дерево папки js
```yml
- js/
    - animation.js - скрипт для работы анимаций (только включения класса animated)
    - cases-script.js - скрипт для работы секции кейсов (только секция кейсов!)
    - init.js - для инициализации компонентов (например инициализация шапки, её цвета и тд)
    - script.js - для общей логики
```

## Pages 

### Описание

Папка содержит основные html-страницы проекта. К этим проектам подключаются основные компоненты и файлы js

### Дерево папки pages/

```yml
- pages/ 
    - cases.html - страница кейсов
    - index.html - главная страница
    - services.html - страница сервисов
```

## Styles

### Описание

Папка содержит основные стили scss, в частности которые относятся к группе компонентов или описывают общие стилевые поведения блоков.

(Папка не содержит стили отдельных компонентов, они содержатся в папке компонента внутри *components/*)

### Дерево папки styles/
```yml
- styles/
    - _animation.scss - файлы анимаций (keyframes + общие паттерны анимирования)
    - _common.scss - стили семантических тегов
    - _fonts.scss - подключение шрифтов через scss
    - _main.scss - основные стили общих классов
    - _media.scss - стили для адаптивной верстки
    - _mixins.scss - миксины для сокращения времени работы
    - _reset.scss - сброс основных стилей браузера
    - _uikit.scss - стили ui-kit
    - _vars.scss - переменные scss
    - style.scss - общий файл, к которому подключаются все стили (даже стили компонентов)
```
