# practice_task

Это рабочий README для лучшей навигации по проекту и понимании того, какие элементы для чего используются 

## Структура проекта

### Дерево проекта
```yml
- public/ - папка скомпилированного проекта
- scr/ - исходники проекта
    - components/ - html компоненты проекта
    - fonts/ - файлы шрифтов
    - images/ - картинки файлов
    - js/ - файлы Javascript для создания логики
    - pages/ - основные html страницы
    - styles/ - общие стили scss
- gulpfile.js - конфигурационный файл для компиляции проекта scr/ в public/
```

### Для компиляции проекта

```bash
gulp build
```
